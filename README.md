# Password Vault (MVP)

Stack:

- Next.js + TypeScript (pages)
- MongoDB
- Server: Next API routes
- Client-side crypto: Web Crypto (PBKDF2 + AES-GCM)

## Quick start

1. npm run dev
2. Open http://localhost:3000
3. http://localhost:3000/login you can login
4. http://localhost:3000/signup you can create an account
5. these are for running the file in localhost
